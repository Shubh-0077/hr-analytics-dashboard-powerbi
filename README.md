# HR Analytics Dashboard (Power BI)

## 📊 Project Overview

This project presents an end-to-end HR analytics dashboard built using real-world employee attendance data. The goal is to analyze employee presence, absence, work-from-home trends , leave patterns etc to generate actionable HR insights.

## 🎯 Objectives

* Track employee attendance trends
* Analyze work-from-home patterns
* Monitor sick leaves and other absences
* Identify workforce availability
* Provide HR decision-making insights

## 🛠 Tools Used

* Power BI (Data Cleaning & Transformation)
* Power BI (Data Modeling)
* Power BI (Dashboard Creation)

## 🔄 Workflow

1. Collected real-world employee attendance dataset
2. Loaded data into Power BI
3. Performed data cleaning and transformation using Power Query
4. Created data model and calculated measures
5. Built interactive HR analytics dashboard
